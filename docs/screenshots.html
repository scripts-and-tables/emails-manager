Screenshots

A walk-through of the app in light mode. The live demo is the canonical version — these are static snapshots.

01 · Landing page
Logged-out view with the hero, features, and signup / login.
02 · Login
Username + password. Successful submit sends an OTP.
03 · Email OTP
6-digit code delivered to the verified email. Resendable.
04 · Dashboard
Welcome hero, three tiles, and a tips card.
05 · Accounts list
Status dots, group labels, Test / Toggle / Edit / Delete actions.
06 · Bulk add
Paste CSV (email, password, host, port). Result summary on submit.
07 · Inbox
Live IMAP. Window / folder / account / group filters across the top.
08 · Email detail
Full body, sender, date. Mark unread / delete hit the IMAP server.

In-app Guide with sticky TOC. Public version.
